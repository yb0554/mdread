# mdread 测试文件

欢迎使用 **mdread** — 轻量级 Markdown 阅读器！

## 功能特性

- ✅ 支持 GitHub 风格 Markdown (GFM)
- ✅ 三套主题: 浅色 / 深色 / 护眼
- ✅ 文件树侧边栏 (懒加载)
- ✅ 便携运行, 无需安装

## 文本格式

**粗体文本**、*斜体文本*、~~删除线~~、`行内代码`
